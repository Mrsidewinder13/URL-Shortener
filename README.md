# Django URL Shortener

## Requirements
- python 3.x
- django 4.x

## Installation
Clone or download this repository.
```
git clone https://github.com/Mrsidewinder13/URL-Shortener.git
```
Then
```
python -m venv .venv
```
```
.venv\Scripts\activate.bat
```
```
pip install -r requirements.txt
```
```
python manage.py runserver
```
